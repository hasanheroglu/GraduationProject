# 3D Survival Simulation Game

For the details check the [thesis](/grad_files/bachelorsthesis.pdf).

## Aim of the project 
Aim of the project is to make every character in the game playable. While doing that, Object-Oriented Programming principles and desgin patterns are used.

## Playable Characters
Right now, there are three playable characters which are human, zombie and cow.

<img src="/grad_files/screenshots/ss_00.png?raw=true" width="600"/>

Every character can have various interactions with the world. Humans can chop tree while cows can't, cows can eat plants and zombies can attack cows.
